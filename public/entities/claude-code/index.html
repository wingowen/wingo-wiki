Claude Code

2026-04-15

laude Code 是 Anthropic 推出的命令行 Agent 工具，用于编程辅助。

核心能力

  • 代码生成与编辑: 理解项目上下文，生成或修改代码
  • Git 操作: 自动提交、创建分支、代码审查
  • 工具使用: 搜索、文件操作、终端命令
  • 安全执行: 权限提示控制，保护系统安全

最佳实践

  • 使用 /bug 命令提交反馈
  • 配置合适的 .claude 项目文件
  • 利用 @ 符号引用项目文件
