再review一下工作流：

```plain text
初始上下文（系统提示词+用户请求）
    ↓[agent loop开始]
    ↓agent读取上下文 → 思考 → 决定行动
    ↓执行工具/行动 → 获得结果
    ↓结果追加到上下文
    ↓[循环继续或结束]
```
