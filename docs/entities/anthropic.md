# Anthropic

Anthropic 公司，致力于构建可靠、可解释和可操控的 AI 系统。

## 核心产品

- **Claude**: AI 助手产品线
- **Claude Agent SDK**: Agent 开发框架
- **Model Context Protocol (MCP)**: 工具集成协议
- **Claude Code**: 编程 Agent 工具
