# 多代理系统

## 分类介绍

多代理系统是由多个相互协作的代理组成的复杂系统，能够解决单个代理难以完成的复杂任务。本分类涵盖了多代理系统的核心概念、协作机制和研究进展，帮助您理解和构建有效的多代理系统。
