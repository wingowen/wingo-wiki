# MCP

## 分类介绍

MCP (Model Context Protocol) 是一种用于管理和传递模型上下文的协议，为代理系统提供了高效的上下文处理能力。本分类涵盖了 MCP 的核心概念、实现方法和最佳实践，帮助您理解和应用 MCP 技术。
