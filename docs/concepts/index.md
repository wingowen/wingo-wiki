# 概念概览

欢迎来到 Wingo Wiki 的概念知识库，这里汇集了 AI 代理、上下文工程、检索增强生成等前沿技术领域的核心概念和实践指南。

## 概念分类

### [Agent Architecture](agent-architecture/)
**代理架构** - 包含代理设计的核心概念、框架理论、实践应用和面试相关内容。

- **核心概念**：代理架构基础、框架理论
- **实践应用**：框架实践、构建有效代理
- **面试相关**：架构设计问题

### [Context Engineering](context-engineering/)
**上下文工程** - 专注于代理上下文管理的核心概念、实践应用和面试相关内容。

- **核心概念**：上下文工程基础、上下文管理
- **实践应用**：有效上下文工程
- **面试相关**：上下文管理问题

### [RAG](rag/)
**检索增强生成** - 涵盖上下文检索、HYDE 技术和 RAG 核心概念，以及相关面试内容。

- **核心概念**：RAG 基础、上下文检索、HYDE 技术
- **面试相关**：RAG 技术问题

### [Tool Use](tool-use/)
**工具使用** - 包括高级工具使用、工具使用基础概念和实践应用。

- **核心概念**：工具使用基础、高级工具使用
- **实践应用**：思考工具、有效工具编写

### [Multi-Agent](multi-agent/)
**多智能体** - 探索多智能体系统的核心概念和实践应用。

- **核心概念**：多智能体基础
- **实践应用**：多智能体研究

### [MCP](mcp/)
**模型上下文协议** - 详细介绍 MCP 的核心概念、代码执行和深度解析。

- **核心概念**：MCP 基础
- **实践应用**：代码执行、深度解析

### [Claude](claude/)
**Claude 相关** - 包含 Claude Agent SDK、代码最佳实践、桌面扩展和事后分析。

- **实践应用**：Agent SDK、代码最佳实践、桌面扩展、事后分析

### [其他](other/)
**其他概念** - 涵盖 LangGraph、提示注入、React 等核心概念，以及各种实践应用。

- **核心概念**：LangGraph、提示注入、React
- **实践应用**：代理技能、超越权限提示、双记忆系统、长期运行代理、斜杠命令、SWE-bench
